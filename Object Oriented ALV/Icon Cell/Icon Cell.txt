DİNAMİK YAPI OLUŞTURMA

en son değerler verebiliyorduk fiyat kısmına ve ortalamanın üstünde ise o fiyat kırmızı eşitse sarı ve ortalamanın altındaysa yeşil verelim.

1-) İconları bul

2-) Form'daki get_total_sum'ı 0dan yazalım.

DATA: lv_total_sum TYPE int4,
      lv_lines TYPE int4,
      lv_avg TYPE int4.


LOOP AT gt_list INTO gs_list.
lv_total_sum = lv_total_sum + gs_list-cost
ENDLOOP.

DESCRIBE TABLE gt_list LINES lv_lines.

lv_avr = lv_total_sum / lv_lines.

LOOP AT gt_list ASSIGNING <gfs_list>.
IF <gfs_list>-cost > lv_avr.
<gfs_list>-durum = '@2C@'.
ELSEIF <gfs_list>-cost < lv_avr.
<gfs_list>-durum = '@23@'.
ELSE.
<gfs_list>-durum = '@1G@'.
ENDIF.
ENDLOOP.
