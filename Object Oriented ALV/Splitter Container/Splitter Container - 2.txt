3-) Top'a gidip

DATA: go_splitter TYPE REF TO cl_gui_splitter_container.

4-) Bu objeyi display alv'ye gidip.

CREATE OBJECT go_splitter (ctrl+space).

5-) Parent,rows ve columns haricini siliyoruz.

Parent = go_cust (hangi objeyi böleceksek onu veriyoruz)
Rows = 1 (bu objeyi kaça bölsün, alt alta duran containerlar)
Columns = 2 (yan yana kaç kolon olsun)

6-) Tekrar Top'a gidiyoruz

go_gui1 TYPE REF TO cl_gui_container.

6-) Geri geliyoruz display_alv'ye ve parçaladığımızın altına

column= 1

container = go_gui1
